# 🏥 Hospital Management API

API REST desenvolvida com Spring Boot para gerenciamento hospitalar.

## 🚀 Funcionalidades

- Pacientes
- Médicos
- Convênios
- Consultas
- Prontuários
- Receitas

## 🧱 Tecnologias

- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database
- Lombok

## 📬 Testes

Collection do Postman disponível em:

📁 /postman/hospital.postman_collection.json

## ▶️ Como rodar

mvn spring-boot:run
